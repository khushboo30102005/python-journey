# Dictionary In Python

## definition :

**Dictionary in python is a built-in data typ that used to store data in Key-Value pairs. Each key is unique and map a value. Dictionaries are unordered, mutable.**
## Creating a Dictionary:
**Dictionaries can be created using curly braces {} or the dict() constructor.**
```python
# Using curly braces
my_dict = {"name": "Alice", "age": 30, "city": "New York"}

# Using the dict() constructor
another_dict = dict(brand="Ford", model="Mustang", year=1964)
```
# Set in Python:

**In Python, a set is an unordered collection of unique elements.**

```python
    my_set = {1, 2, 3, 4, 5}
    print(my_set)
    # Output: {1, 2, 3, 4, 5}
```
